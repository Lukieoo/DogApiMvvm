# DogApiMvvm
Short Example Mvvm

 
 
## Technologies
Project is created with:

* Depedency injection with Hilt  
* RxJava 3
* MVVM
* simple UI/UX 
* Kotlin 
* Room
* Rest Api https://dog.ceo/dog-api/documentation/
